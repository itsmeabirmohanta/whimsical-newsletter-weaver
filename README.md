## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/fsl-newsletter-builder.git
   cd fsl-newsletter-builder
   ```

2. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   ```
